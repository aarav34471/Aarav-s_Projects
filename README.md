# AI & Machine Learning Projects

Computer Science undergraduate interested in machine learning systems, financial modelling, and AI methods that work under noisy or uncertain real-world conditions.

## Featured Projects

### Behavioural Bias Detection
Transformer-based modelling on trading time-series data using weak supervision, self-supervised learning, and explainability methods.

### Macroeconomic Event Impact Modelling
Financial NLP and market-data project using FinBERT embeddings, macroeconomic release variables, volatility features, and classifiers such as Logistic Regression, XGBoost, and MLP.

### Neural Network Optimisation
Benchmarking optimisation methods including SGD, Genetic Algorithms, PSO, Differential Evolution, and related approaches for neural network training.

### CNN for Breast Cancer Classification
Computer vision project using CNNs on BreastMNIST, with preprocessing and hyperparameter tuning.

### Parallel Computing
CUDA-based implementations and benchmarking against CPU baselines.

### Full-Stack Job Portal
Django REST and React application with authentication, role-based flows, and backend API design.
